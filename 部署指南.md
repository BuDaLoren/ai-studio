# 部署到 Vercel 指南

## 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com
2. 点击右上角 "+" → "New repository"
3. 仓库名：ai-studio
4. 选择 Public 或 Private
5. 点击 "Create repository"

## 步骤 2：推送代码

创建仓库后，复制并执行以下命令：

```bash
git branch -M main
git remote add origin https://github.com/BuDaLoren/ai-studio.git
git push -u origin main
```

## 步骤 3：部署到 Vercel

### 方法 A：网页版部署（最简单）

1. 访问 https://vercel.com
2. 点击 "Sign Up" 或 "Login"，使用 GitHub 账号登录
3. 点击 "Add New Project"
4. 选择刚才创建的 ai-studio 仓库
5. 点击 "Deploy"

等待 1-2 分钟，网站就部署好了！

### 方法 B：使用 CLI

```bash
npm install -g vercel
vercel login
vercel
```

## 步骤 4：访问网站

部署完成后，Vercel 会给你一个域名，比如：
- https://ai-studio.vercel.app

## 更新网站

以后修改代码后，只需：

```bash
git add .
git commit -m "更新说明"
git push
```

Vercel 会自动重新部署！

## 自定义域名（可选）

1. 在 Vercel 项目设置中点击 "Domains"
2. 添加你的域名
3. 按提示配置 DNS 记录

## 🚀 国内访问加速方案

由于 Vercel 在中国大陆访问可能较慢或不稳定，以下是几种解决方案：

### 方案一：使用 Cloudflare CDN 加速（推荐）

1. **注册 Cloudflare 账号**：访问 https://cloudflare.com
2. **添加网站**：输入你的域名，选择免费计划
3. **更改 DNS 服务器**：按照 Cloudflare 提示，将域名的 DNS 服务器改为 Cloudflare 提供的地址
4. **在 Cloudflare 中配置**：
   - 进入 DNS 设置，确保域名指向 Vercel（CNAME 记录指向 `cname.vercel-dns.com`）
   - 开启 CDN 加速和 SSL/TLS 加密
   - 启用 "Always Use HTTPS"
5. **在 Vercel 中配置域名**：将自定义域名指向你的域名

### 方案二：部署到 Cloudflare Pages（更稳定）

1. **在 GitHub 上推送代码**
2. **访问 Cloudflare Pages**：https://pages.cloudflare.com
3. **连接 GitHub 仓库**
4. **配置构建设置**：
   - 构建命令：`npm run build`
   - 输出目录：`dist`
   - 环境变量：添加 EmailJS 相关变量
5. **点击部署**：Cloudflare Pages 在中国访问更稳定

### 方案三：使用国内托管服务

1. **阿里云 OSS + CDN**：
   - 将 `dist` 目录上传到 OSS
   - 开启静态网站托管
   - 配置 CDN 加速
2. **腾讯云 COS + CDN**：类似阿里云方案

## 📧 邮件发送功能配置

网站的联系表单使用 EmailJS 发送邮件，需要配置以下步骤：

### 1. 注册 EmailJS 账号
- 访问 https://www.emailjs.com
- 点击 "Sign Up" 免费注册

### 2. 创建电子邮件服务
- 进入 "Email Services" 页面
- 点击 "Add New Service"
- 选择你的邮箱提供商（如 Gmail、QQ邮箱、Outlook等）
- 按照提示授权 EmailJS 访问你的邮箱

### 3. 创建邮件模板
- 进入 "Email Templates" 页面
- 点击 "Create New Template"
- 使用以下变量创建模板：
  - `{{from_name}}` - 用户姓名
  - `{{from_email}}` - 用户邮箱
  - `{{phone}}` - 用户电话
  - `{{service}}` - 感兴趣的服务
  - `{{message}}` - 留言内容
  - `{{date}}` - 提交时间
- 保存模板并记录 **Template ID**

### 4. 获取 API 密钥
- 进入 "Account" 页面
- 找到 **Public Key**（以 `public_` 开头）
- 找到 **Service ID**

### 5. 配置环境变量
在项目根目录创建 `.env.local` 文件（不要提交到 Git），内容如下：

```env
VITE_EMAILJS_SERVICE_ID=你的_service_id
VITE_EMAILJS_TEMPLATE_ID=你的_template_id
VITE_EMAILJS_PUBLIC_KEY=你的_public_key
VITE_RECEIVER_EMAIL=rendaloren@outlook.com
```

### 6. 在部署平台添加环境变量
- **Vercel**：项目设置 → Environment Variables
- **Cloudflare Pages**：设置 → Environment Variables
- **Netlify**：Site settings → Environment variables

## 🔧 本地开发

1. 安装依赖：`npm install`
2. 复制环境变量文件：`cp .env.example .env.local`
3. 填写正确的 EmailJS 配置
4. 启动开发服务器：`npm run dev`
5. 访问 http://localhost:3000
